## 📝 Projects Overview

### Task 1: Student Grade Tracker
**Description:**
The Student Grade Tracker is a Java application used to manage student marks and analyze their performance. The program collects student grades and calculates statistics such as average, highest, and lowest scores.

**Features:**
- Enter student grades
- Calculate average marks
- Find highest score
- Find lowest score
- Display grade summary

**Concepts Used:**
- Arrays / ArrayList
- Loops
- Conditional statements
- Basic Java I/O

---

### Task 2: Stock Trading Platform
**Description:**
The Stock Trading Platform is a simulation program that demonstrates a simple stock market trading environment. Users can view stock information, perform buy/sell operations, and track portfolio performance.

**Features:**
- Display stock market data
- Buy stocks
- Sell stocks
- Track portfolio value
- Simulate trading operations

**Concepts Used:**
- Object-Oriented Programming (OOP)
- Classes and Objects
- Methods
- Data handling

---

### Task 4: Hotel Reservation System
**Description:**
The Hotel Reservation System is a Java-based application used to manage hotel room bookings. Users can search for available rooms, book rooms, cancel reservations, and view booking details.

**Features:**
- Room categorization (Standard, Deluxe, Suite)
- Book hotel rooms
- Cancel reservations
- View booking details
- Payment simulation
- Store booking data using file handling

**Concepts Used:**
- Object-Oriented Programming (OOP)
- File handling
- Classes and Objects
- User input handling
